## 提出結果

```python
def getInt():
    return int(input())


def getIntList():
    return list(map(int, input().split()))


def main():
    N = getInt()
    A = [0] + getIntList() # 先頭にダミーを入れてインデックスを合わせる
    X = getInt()

    print(A[X])


if __name__ == "__main__":
    main()
```
