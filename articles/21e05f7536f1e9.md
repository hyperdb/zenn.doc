## 提出結果

```python
def getIntList():
    return list(map(int, input().split()))


def main():
    d = getIntList()
    d.sort()

    if d[2] == d[0] + d[1]:
        print('Yes')
    else:
        print('No')


if __name__ == "__main__":
    main()
```
