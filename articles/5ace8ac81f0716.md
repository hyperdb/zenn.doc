- 🔗[ABC-454 B - Mapping](https://atcoder.jp/contests/abc454/tasks/abc454_b)

## 提出結果

```python
def getIntMap():
    return map(int, input().split())


def getIntList():
    return list(map(int, input().split()))


def main():
    N, M = getIntMap()
    F = getIntList()

    # 設問 1
    print("Yes" if len(set(F)) == len(F) else "No")
    # 設問 2
    print("Yes" if len(set(F)) == M else "No")


if __name__ == "__main__":
    main()
```
